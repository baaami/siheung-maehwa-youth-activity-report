# PLAN.md

## 목표

- 학생이 파일 기반 저장소에 동아리 활동일지를 작성, 수정, 제출할 수 있는 기능 구현
- 관리자가 동아리와 활동일지를 관리하고 제출된 활동일지를 PDF 및 사진으로 다운로드할 수 있는 기능 구현

## 현재 우선순위

1. 파일 기반 MVP 저장/조회/수정 흐름 안정화
2. 학생/관리자 보고서 편집 UX 다듬기
3. Playwright PDF 출력 품질과 잘림 감지 정확도 보완
4. 사진 저장 구조를 이후 DB/외부 스토리지 전환 가능하게 유지
5. 이후 Prisma/PostgreSQL 전환 시 데이터 계층만 교체하도록 정리

## 다음 작업 후보

- 학생/관리자 화면 저장 성공 후 토스트나 저장 완료 피드백 추가
- 업로드 사진 미리보기 URL 해제 처리와 파일 교체 UX 개선
- 관리자 활동일지 목록에 제목 검색 추가
- PDF 템플릿 미세 조정과 잘림 경고 문구 보완
- 파일 저장소를 Prisma 저장소로 교체할 때 필요한 어댑터 계층 설계

## 미정 또는 추후 작업

- 실제 로그인/인가 도입
- 수정 이력 관리
- PDF 작업 큐 분리
- Google Cloud Storage 같은 외부 파일 스토리지 전환
- Prisma/PostgreSQL 기반 영속성 전환
